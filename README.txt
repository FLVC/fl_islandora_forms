flvc_islandora_xml_builder_forms
================================

Exported XML Formbuilder files for islandora.
Forms are generated by formbuilder and output as XML. 
They must be uploaded through islandora xml formbuilder gui as a new form
to update.

-

---Full Mods---
Primary MODS entry form. Fields, mapping, and labels approved by ISG. Not exhaustive.

---MODS Simple Entry---
Abridged MODS entry form.

---DC-Tagged MODS---
Dublin Core labeled fields mapped to MODS.


---Full Mods_Newspaper---
Full MODS customized for newspaper entry
